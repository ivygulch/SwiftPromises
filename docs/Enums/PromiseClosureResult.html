PromiseClosureResult

public enum PromiseClosureResult<T>

PromiseClosureResult is returned by promise fulfill and reject closures so they can return a type-safe value, an error, or a promise to be used in the promise chain. This allows either type of closure to pass along a similar value (e.g. type-safe value -> type-safe value) or redirect the dependent, chained promises into an alternative path by returning a type-safe value from a reject closure.

  • .Value - it will cause any dependent promises to be fulfilled with the associated type-safe value
  • .Error - it will cause any dependent promises to be rejected with the associated error value
  • .Pending - it will chain this instance’s resolution to the associated promise’s resolution
  • Error

    Associated value is required and is the error that will trigger the reject closure for a promise
